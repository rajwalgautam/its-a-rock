# v0.4.0

A major release redesigning the climbing form, improving date handling, and enhancing the climb tiles with better visual feedback.

## What's new

**New climbing features & form redesign**
- Completed checkbox replaces the Project/Sent toggle for clearer intent
- Smart date picker with calendar interface — "Completed" date field only shows when a climb is marked as completed
- Date picker displays "Today" when selecting the current day, and the formatted date for other days
- Location autocomplete with searchable dropdown for previously used gyms
- Photo picker integrated into the floating action button for quicker access

**Enhanced climb visualization**
- Status indicator dots on climbing tiles (green for completed, gray for incomplete)
- Climb date display on tiles showing "Jun 13" for current year and "Jun 13 2025" for past years
- Improved column density control with clearer labels: "Large" (2 columns), "Medium" (3 columns), and "Small" (4 columns)

**Improved menus**
- Long-press menu now shows "Mark as completed" and "Mark as incomplete" instead of old "Mark as sent/project" terminology

## Under the hood

- New `LocationPickerField` component with Zustand state management for location history
- Refactored `DatePickerModal` component with improved cross-platform date handling
- Added `formatShortDate()` utility for consistent date formatting on tiles
- Updated `FloatingAddButton` with custom popup menu and photo selection logic
- Enhanced `PhotoPickerField` with optional `hideButtonsWhenSelected` prop for cleaner form layout
