# v1.0.0

It's A Rock reaches 1.0 — and corrects a versioning hiccup from the last build.

## What's new

- **Version fix** — the previous release shipped showing its version as "v0.50"
  instead of "v0.5.0". That malformed number also broke the in-app update check,
  which stopped recognizing newer releases. This release sets the version to
  1.0.0 and restores update detection, so "Check for Updates" works again.
- **1.0 milestone** — the first stable release. It includes everything from the
  0.x series: a full-screen photo viewer with zoom, photo crop, grade ranges,
  History filter & sort, and gym/location management.

Updating from the last build makes no functional changes — your climbs, photos,
and locations are all preserved.

## Under the hood

- No application code changed since the previous build; this release exists to
  correct the app version string (0.50 → 1.0.0) and restore update detection.
  `isNewerVersion` parses versions into numeric segments, so the malformed
  "0.50" ([0, 50]) outranked any real "0.x.y", which is why no 0.x release would
  have been offered as an update.
- Follow-up #94 will make the release workflow reject a version with no matching
  `changelogs/vX.Y.Z.md` (leading `v` included) before cutting the tag, so a typo
  like "v0.50" fails fast instead of shipping.
