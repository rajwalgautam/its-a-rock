# v1.5.2

Jot down a note without hunting for the Edit button first.

## What's new

**Climb card**
- **Add note button** — the climb detail card now has an **Add note** button, sitting just above your existing notes in the same style as **Edit**. Tap it to jump straight into the editor with a fresh, empty note ready to fill in.

## Under the hood

- `RouteCard` gained an **Add note** action styled like the existing **Edit** button; both now route through a shared `startEditing(withNewNote)` helper that opens `RouteForm` in edit mode.
- `RouteForm` takes a new optional `startWithNewNote` prop; when set, `toState` appends one empty draft (via the now-exported `emptyDraft` helper from `NotesEditor`) so the editor opens with a blank note card already added.
