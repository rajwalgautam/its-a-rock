# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report privately via GitHub's
[private vulnerability reporting](https://github.com/rajwalgautam/its-a-rock/security/advisories/new)
(Security tab → "Report a vulnerability"). This is enabled for this repository.

You can expect an initial response within a reasonable time. If a fix is
warranted, it will be coordinated through a private security advisory before
public disclosure.

## Supported versions

This project is pre-1.0; only the latest `main` is supported.
